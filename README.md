# pyEulerCurves  
A python package to compute Euler Characteristic Curves of point-cloud or image data with an interface Ã  la scikit-learn. 

## Prerequisites

* A compiler with C++11 support
* pybind11
* numpy
* scikit-learn

## Installation

Just clone this repository and pip install it.

```bash
git clone https://github.com/dgurnari/pyEulerCurves.git
pip install ./pyEulerCurves
```
